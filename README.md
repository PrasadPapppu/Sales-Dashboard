# Tableau Dashboard Project

This repository contains a Tableau dashboard project focused on sales and customer analytics. The dashboard is built using Tableau Desktop Public Edition and leverages the included CSV data files for all analysis.

## Features

- **Sales Dashboard**: 
  - Analyzes total sales, profit, and quantity by month using provided data.
  - Visualizes sales and profit by subcategory.
  - Tracks KPIs and business trends over time.

- **Customer Dashboard**:
  - Examines customer metrics: total customers, average sales per customer, total orders.
  - Highlights top customers by profit and distribution by number of orders.

## Folder Structure

- `/data/`: Contains the CSV files used as data sources for the dashboards.
- `/icons/`: Visual assets and icons for dashboard interactivity.
- `mockup.pdf`, `project_phases.pdf`: Documentation of design process and dashboard mockups.
- `dashboard.twbx`: Tableau workbook file containing all dashboard logic.

## Project Phases

1. **Requirements Analysis**: Gathered business requirements, chose chart types, sketched mockups, and planned dashboard UI.
2. **Data Source Setup**: Connected and prepared CSV files, performed data modeling and cleaning.
3. **Chart Building**: Created calculated fields, built charts, formatted tooltips and axes for clarity.
4. **Dashboard Design**: Assembled all charts, added filter and icon interactivity, formatted layouts for usability (details in PDFs).

## Usage

1. Download or clone this repository.
2. Open `dashboard.twbx` in Tableau Desktop Public Edition.
